Question 1 10 marks
Merge Sort with Performance Metrics

Implement merge sort from scratch and instrument it to report internal performance data.

Input
  • Integer n
  • Array of n integers
Output
  • Sorted array
  • Comparison count
  • Merge operation count
Tasks
  1. Implement merge sort manually — no library sort functions.
  2. Print the sorted array.
  3. Print the total number of element comparisons made.
  4. Print the total number of merge operations performed.
1 ≤ n ≤ 10⁵ integers may be negative
Question 2 10 marks
0/1 Knapsack — DP Table & Item Recovery

Solve the 0/1 knapsack problem using dynamic programming and reconstruct the exact set of chosen items.

Input
  • Integer n — number of items
  • Integer W — capacity
  • Array weight[n]
  • Array value[n]
Output
  • Maximum achievable value
  • Indices of selected items
Tasks
  1. Build the full DP table of size (n+1) × (W+1).
  2. Compute and print the maximum value.
  3. Backtrack through the DP table to recover and print the selected item indices.
1 ≤ n ≤ 100 1 ≤ W ≤ 10⁴
Question 3 10 marks
Binary Search — Multi-Query with Comparison Count

Apply binary search across multiple queries on a pre-sorted array and track total comparisons.

Input
  • Integer n
  • Sorted array of n integers
  • Integer q — query count
  • q target values
Output
  • 0-based index per query, or -1 if not found
  • Total comparisons across all queries
Tasks
  1. For each query, perform binary search on the sorted array.
  2. Print the 0-based index of the target, or -1 if absent.
  3. After all queries, print the cumulative comparison count.
1 ≤ n ≤ 10⁵ 1 ≤ q ≤ 10³
